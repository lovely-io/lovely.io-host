STL Ajax Module

This package provides the standard ajax functionality

Usage

Install it locally with lovely install ajax or hook up from http://cdn.lovely.io

Lovely(['ajax'], function(ajax) {
  ajax.load('/my.url', {
    success:  function() { /* ... */ },
    failure:  function() { /* ... */ },
    complete: function() { /* ... */ }
  });
});
